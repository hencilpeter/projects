1.  Env setup
	(i) subst X: "C:\Users\User\Documents\GitHub\PythonProjects"

		py -m venv marania_invoice_venv

		\marania_invoice_venv\Scripts\activate

		pip install django
		pip install psycopg2
		pip install Pillow 
		pip install uuid
		pip install requests
		pip install django-crispy-forms
		pip install num2words
		pip install django-widget-tweaks

		pip install xhtml2pdf --target .\Lib\site-packages
		
2. Project/Application Setup 	
	(i) Create a project 
		django-admin startproject marania_invoice_proj
    (ii) create application 
		python manage.py startapp marania_invoice_app 
3. Run the server 
		python manage.py runserver
		Try the below:
		http://127.0.0.1:8000/
		http://127.0.0.1:8000/admin/
		py manage.py runserver 0.0.0.0:5000  #  for specific port

	(i) create default database and tables : 
	       python manage.py makemigrations
	(ii) python manage.py migrate
	(iii) create super user  : 
	      python manage.py createsuperuser

Key Automation Ideas
Automatic Stock Updates

When:

Purchase received → Stock increases.
Production consumes twine → Stock decreases.
Stock adjustment → Stock updated.
Automatic Invoice Balance
Invoice Total
- Payments Received
= Outstanding Balance
Automatic P&L

Generate monthly P&L dynamically from transactions instead of storing profit values.

Low Stock Alerts

Trigger notification when:

Current Stock < Reorder Level
Dashboard KPIs

Show:

Today's Sales
Outstanding Receivables
Outstanding Payables
Current Twine Stock
Low Stock Items
Monthly Profit
Top Customers
Top Selling Net Types

This structure will scale well from a small fishnet factory to a multi-factory operation and keeps inventory, costing, and profitability fully connected.
