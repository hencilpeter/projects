Marania Invoice- Application Development
========================================
1.  Env setup
	(i) subst X: "C:\Users\User\Documents\GitHub\PythonProjects"
	(ii) virutal environment 
		py -m venv marania_invoice_venv
	(iii) activate virtual environment 
		\marania_invoice_venv\Scripts\activate
	(iv) install django and libs 
		pip install django
		pip install psycopg2
		pip install Pillow 
		pip install uuid
		pip install requests
		pip install django-crispy-forms
    
	
2. Project/Application Setup 	
	(i) Create a project 
		django-admin startproject marania_invoice_proj
    (ii) create application 
		python manage.py startapp marania_invoice_app 
3. Run the server 
		python manage.py runserver
		Try the below:
		http://127.0.0.1:8000/
		http://127.0.0.1:8000/admin/
		py manage.py runserver 0.0.0.0:5000  #  for specific port 
4. Create Super user 
	(i) create default database and tables : 
	       python manage.py makemigrations
	(ii) python manage.py migrate
	(iii) create super user  : 
	      python manage.py createsuperuser    
		(give user name(email id )  and password ) 
	     Username : jhpeter@gmail.com (user) 
         Password: Test1234	
5. settings change 
   - timezone change:  settings.py
          ALLOWED_HOSTS = ['*']
          TIME_ZONE = 'Asia/Kolkata' #'UTC'
		  STATIC_ROOT = os.path.join(BASE_DIR, 'static/')
		  MEDIA_ROOT = os.path.join(BASE_DIR, 'uploads')
	      MEDIA_URL = '/uploads/'
  - register the app 
  INSTALLED_APPS = [
    ...
    'marania_invoice_app',]
6. create the required models 
	- Customer
	- Product
	- ProductPrice
	- Configuration
	- CustomerPriceMap	
7. Register the models in admin.py 
	admin.site.register(Client)
	admin.site.register(Product)
	admin.site.register(Invoice)
	admin.site.register(Settings)
8. Changes for adding forms:
   - create a file forms.py  in marania_invoice_app
   - inside the project root create "tempaltes" folder and include the path in settings.py: TEMPLATES []
             also create a project directory inside and keep the html files 
       marania_invoice_proj\templates\marania_invoice_app
	  settings.py:
	  ------------
		'DIRS': [os.path.join(BASE_DIR,'templates')],
	- inside the application folder, create a views.py & urls.py file and copy the code 
	- register the crispy forms in settings.py  :  'crispy_forms',
